# 🎬 Netflix Clone  

A simple **Netflix UI clone** built with **HTML, CSS, and Bootstrap**, replicating the homepage design of Netflix.  

## 🚀 Features  
✅ Responsive Design  
✅ Trending Movie Section  
✅ FAQ Section  
✅ Language Selection & Sign-in Button  
✅ Hover Effects on Movie Thumbnails  

## 🛠️ Tech Stack  
- HTML  
- CSS  
- Bootstrap  
- FontAwesome
